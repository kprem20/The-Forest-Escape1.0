## 📌 Project Overview

**The Forest Escape** is an interactive story-based adventure game built using **Python (Tkinter)** and an enhanced **Web version (HTML, CSS, JavaScript)**.

The game allows players to explore a mysterious forest where every choice changes the direction of the story. It includes multiple difficulty levels (Easy, Medium, Hard), branching paths, multiple endings, sound effects, and optional voice narration by a guide character named **Aiden**.

This project combines storytelling, GUI design, and programming logic to create an engaging and educational experience.

---

## 🎮 Features

### 👤 User Features
- Choice-based story progression
- Multiple difficulty modes
- Multiple endings (Win / Lose / Neutral)
- Animated typing text effect
- Background music and sound effects
- Voice narration support
- Replay option
- Progress tracking system

### ⚙️ System Features
- Story stored in external `story_data.json`
- Modular and clean code structure
- Scene-to-scene navigation logic
- Separate Python and Web versions
- Lightweight and runs on low-end systems
- Easy to expand with new scenes and endings
